Personal Website

I built this site using html, css, JQuery, and Bootstrap.

I learned html, css, and a bit of JQuery by following intro tutorials on Codecademy. After making a mock website using only a text editor, I felt comfortable enough with the languages to start on my site.

This site uses Bootstrap for the JavaScript and base classes for grid layout and spacing. I have added customizations to fit my specific needs.

I've also written a few JQuery functions of my own to animate things like the turquoise ring which appears when the cursor hovers over menu options.

The Best of Wikipedia

A Chrome extension

Have you ever been frustrated that the "random article" link in Wikipedia often chose short articles without any real content? The Best of Wikipedia is a Chrome extension which solves this problem.

I recently discovered a link which randomly chooses a Wikipedia article with actual content. The issue was that you can't save the link as a bookmark because it doesn't refer to any specific page. My extension uses this link to choose a random article with good content, solving the problem.
